# 🔌 API Endpoints

---

## 🔎 Search API

### Endpoint

```http
GET /api/search?q=oasis
```

### Description

Search music/videos with realtime metadata.

---

## 🎵 Play API

### Endpoint

```http
GET /api/play?query=alan walker
```

### Description

Return playable audio response.

---

## 🕒 Waktu Sholat API

### Endpoint

```http
GET /api/waktusholat?zone=wib
```

### Description

Get prayer schedules.

---

## 🔳 QR Generator API

### Endpoint

```http
GET /api/qr?text=hello
```

### Description

Generate QR image from text.

---

## 🐙 GitHub Stalk API

### Endpoint

```http
GET /api/githubstalk?user=torvalds
```

### Description

Fetch GitHub profile information.

---

## 🖼️ CPhoto API

### Endpoint

```http
GET /api/cphoto?text=anime
```

### Description

Generate AI image response.
